# ISFP-Connect

ISFP-Connect 官方桌面客户端是一款专为ISFP连飞平台设计的综合性辅助工具。基于 **PySide6** 框架开发，提供极佳的 Windows 原生性能与现代化视觉体验。

## 🌟 核心功能

- **🚀 现代化首页**: 实时展示 ISFP 连飞服务器状态、在线机组人数及网络延迟。采用极简航空仪表风格设计。
- **🌤️ 气象报文查询**: 支持全球机场 METAR 与 TAF 报文查询。报文经过 HTML 格式化美化，清晰易读。
- **👥 在线机组监控**: 实时获取连飞服务器上的飞行员动态，包含呼号、高度、速度、航路及机型信息。
- **📝 本地飞行计划制作**:
  - **自动识别**: 输入飞机注册号（如 B-32DN）自动获取高清飞机照片及机型。
  - **航迹预览**: 内置 SkyVector 交互式地图，根据输入的起降机场自动渲染实时航线。
  - **分栏布局**: 左侧表单制作，右侧地图预览，支持自由调节比例。
- **✈️ X-Plane 连飞支持**:
  - **原生插件**: 通过 X-Plane TCP 插件实时获取飞行数据。
  - **插件管理**: 自动检测 X-Plane 安装路径，支持一键安装/卸载插件。
  - **FSD 协议**: 完整支持 FSD 协议，可连接连飞网络。
  - **实时位置**: 自动将 X-Plane 位置数据发送到连飞服务器。
- **🔔 灵动岛通知**: iPhone 风格的悬浮通知组件，显示服务器消息、航班状态等，支持丝滑动画效果。
- **📡 服务器消息**: 实时接收服务器广播消息，带提示音和灵动岛展示。

## 🛠️ 安装要求

- **操作系统**: Windows 10/11 (推荐)
- **Python 版本**: 3.9+
- **X-Plane**: 11 或 12
- **核心依赖**:
  - `PySide6`: GUI 框架
  - `PySide6-WebEngine`: 地图渲染内核
  - `requests`: 网络请求

## 🚀 快速开始

1. **克隆/下载项目**
2. **安装依赖**:
   ```powershell
   pip install -r requirements.txt
   ```
3. **准备素材**:
   确保 `assets/` 文件夹下包含：
   - `logo.png`: 应用图标
   - `background.png`: 全局背景图
4. **运行程序**:
   ```powershell
   python main.py
   ```

## ✈️ X-Plane 连飞设置

1. **选择 X-Plane 路径**:
   - 打开设置页面，在 "X-Plane 插件管理" 区域点击 "选择 X-Plane 路径" 或 "自动检测"。

2. **安装插件**:
   - 点击 "安装插件" 按钮，自动将 ISFP Connect 插件安装到 X-Plane。

3. **连接 X-Plane**:
   - 启动 X-Plane，进入飞行。
   - 在 ISFP-Connect 中点击 "连接 X-Plane" 按钮。

4. **连接 FSD 服务器**:
   - 输入呼号、CID、密码和服务器地址。
   - 点击 "连接服务器" 按钮开始连飞。

## 📦 打包教程

推荐使用 **Nuitka** 进行高性能打包，以确保任务栏图标正常显示：

1. **安装 Nuitka**:
   ```powershell
   pip install nuitka
   ```
2. **执行打包命令**:
   ```powershell
   nuitka --standalone --show-progress --plugin-enable=pyside6 --windows-disable-console --include-data-dir=assets=assets main.py
   ```

## 🛠️ 技术细节

- **AppUserModelID**: 解决了 Python 程序在 Windows 任务栏无法显示独立图标的顽疾。
- **异步多线程**: 所有 API 请求（气象、图片下载、在线人数）均在独立线程执行，确保 UI 界面永不卡顿。
- **圆角裁剪算法**: 动态从云端下载飞机图片并实时进行圆角裁剪处理，保持视觉风格统一。
- **灵动岛动画**: 使用 QPropertyAnimation 实现丝滑的展开/收起动画效果。
- **FSD 协议**: 完整实现 FSD 协议，支持位置报告、文本消息、心跳等功能。
